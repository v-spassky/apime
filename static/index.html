apime.app beta

This is an image classification API
for predicting if a given picture is anime or not.

Currently the model has about 85% accuracy.

Send an HTTP POST request to the /is_it_anime route
with the 'pfp-url' header containing a link to the image of interest.

The answer will be either 200 { "conclusion": "Yes"/"No" }
or 400/500 { "details": "Error message..." }.
